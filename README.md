# Tech Matrix Agent — Release Distribution

This repository hosts release binaries for the **Tech Matrix Agent** desktop application.

## Download

Customers should download via the website:

**https://techmatrix.ai/agent/**

The download buttons there 302-redirect to the matching release asset on this repository — same URL pattern, faster CDN.

## Verification

Each release page lists SHA-256 hashes for every artifact. Verify after download:

- Windows: `Get-FileHash <file>` (PowerShell)
- Linux:   `sha256sum <file>`

## Source code

This repository contains **release artifacts only**. Source code lives in a separate private repository and is not publicly accessible.
